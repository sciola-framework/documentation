Installation

There are two ways to install the Sciola Framework:

This first installation method is recommended.

debian@debian:~$ npx sciola install

Or this other way:

1. Download the framework here and unzip it in your development directory.

Example: /var/www/sciola

2. Go to the packages directory and run the following command to install:

LINUX

debian@debian:~$ cd /var/www/sciola/packages

debian@debian:/var/www/sciola/packages$ npm install sciola

Note

Give write permission to the directory: /var/www/sciola/writable

Built-in web server

debian@debian:~$ npx sciola start php -d app/root

or

debian@debian:~$ cd app/root

debian@debian:~/app/root$ npx sciola start php

Access the application through the url configured in the config/server.ini file. Example: http://localhost:8000
