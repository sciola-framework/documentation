Installation

Download the framework here and unzip it in your development directory.

Example: /var/www/sciola

Go to the packages directory and run the following command to install:

LINUX

debian@debian:~$ node /var/www/sciola/packages/install

Give write permission to the directory: /var/www/sciola/writable

Access the application through the url that is configured on your web server. Example: http://localhost
